## 🔧 How to Add and Use Snippets

Poof works by letting you create shortcuts that expand into longer text. Here’s how to add snippets:

1. Click the `Add Snippet` or `+` button inside Poof.
2. In the shortcut field, type a short code you want to use (like `addr` for your address).
3. In the content field, type the full text you want this shortcut to expand to.
4. Save the snippet.

To use a snippet, just type the shortcut anywhere in your PC (in documents, emails, browsers, etc.), and Poof will replace it with the full text.

## ⚙ Poof Settings You Might Want to Adjust

Poof gives you simple options to fit your needs:

- **Auto-start with Windows:** Enable this if you want Poof to run automatically when your PC boots.
- **Trigger key:** Choose which key you press after a shortcut to expand it (for example, space or enter).
- **Sync snippets:** Save your snippets to a cloud service if you want to use them on multiple devices.
- **Notifications:** Turn on or off pop-up alerts when a snippet expands.

---

## 🐞 Troubleshooting and Tips

- If snippets do not expand, check if Poof is running in the background. You can see it in the system tray near the clock.
- Make sure the shortcut you type exactly matches the snippet shortcut you created.
- Restart Poof if it stops responding.
- If you see no installer file for Windows on the release page, look for a `.zip` file and follow included instructions.

---

## 🔒 Privacy and Data

Poof only saves your snippets locally on your device unless you choose cloud sync. Your text snippets are not shared with others.

---

## 📂 Uninstalling Poof

To remove Poof:

1. Open Windows Settings.
2. Go to `Apps` > `Apps & features`.
3. Find `Poof` in the list.
4. Click it and select `Uninstall`.

Follow on-screen prompts to complete the uninstall.
